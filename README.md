# Tower of Hanoi - Java

Implementação do problema clássico Torre de Hanoi utilizando Java.

## Sobre
Projeto desenvolvido para praticar lógica de programação, estruturas básicas e recursividade.

O programa permite:
- mover discos entre as torres
- visualizar o estado das torres
- reiniciar o jogo
- contar o número de movimentos

## Regras do jogo
1. Apenas um disco pode ser movido por vez  
2. Um disco maior não pode ficar sobre um menor  
3. São utilizadas três torres
